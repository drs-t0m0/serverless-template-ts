# Serverless Framework Typescript Template

## Required

- Node.js
- Serverless Framework
  - https://www.serverless.com/

```
$ npm install -g serverless
```

## Plugin

- https://github.com/serverless-heaven/serverless-webpack

## Deploy

```
$ sls deploy
```

## Test

```
$ sls invoke -f hello
```

```
$ sls invoke local -f hello
```

## Remove

```
$ sls remove
```
